SVG Star

Flexbox has been used as there are only two div's in need of styling with position and layout (those containing the SVG and Canvas Elements). CSS Grid would have been more appropriate with several other div's - with need of styling with both rows and columns - source. In short, flexbox was regarded as the preferred (most effective) way to style this webpage.

Canvas is drawn in an external js-file. jQuery is used to invoke actions on the Canvas, such as hover. jQuery also toggles this text.
SVG is defined inline in the HTML due to the simplicity of the element. The interactive part is ensured by a simple css-command. This has the advantage of targeting the polygon and circles themselves, instead of the entire element. jQuery is imported in the script tag below, and used in index.js to simplify all the actions (.ready and .hover) on the elements in the html-file.

The website is tested in both Mozilla Firefox and Google Chrome - two of the most used browsers. Functionality is tested, in addition to checking the console in each browser for errors or warnings.
