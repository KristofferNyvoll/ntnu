Questions

  1. The servers manage the files and connect directly to the network. They handle requests and reply with the correct adress
  2. The client is the user. He/she sends requests for a certain domain/adress. That request is handled by the servers and the client is redirected to the requested site.
  3. IP is short for Internet Protocol. The IP adress is unique for every computer, as is required for communication over the network. It can be regarded as a mail adress, because the mail cannot be delivered unless the mailbox has a valid adress. The same principle goes for IP adresses.
  4. DNS is short for Domain Name System. It contains the names of the domains (aka websites) and their respective IP adresses. In a way, it is similar to a phonebook, connecting names and numbers
  5. The world wide web is an information sharing model using the HTTP protocol.It is often confused with the internet, but the internet is a masive networking infrastructure.
  6. HTML, or Hypertext markup language, is the publishing language of the www. It handles the document structure and formatting.
